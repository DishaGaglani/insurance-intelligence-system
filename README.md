# Insurance Intelligence System

## Overview
This project builds an end-to-end insurance analytics system that integrates multiple datasets to:
- Predict customer risk
- Estimate claim loss
- Detect fraudulent behavior
- Optimize insurance premium pricing

## Datasets Used
- Medical Cost Dataset (Customer Data)
- Allstate Claims Severity (Claims Data)
- Weather Dataset (External Risk)

## Features
- Risk Segmentation (Clustering)
- Claim Probability Prediction
- Loss Estimation (Regression)
- Fraud Detection (Anomaly Detection)
- Premium Optimization Engine

## Tech Stack
- Python (pandas, sklearn, xgboost)
- Streamlit
- Power BI / Tableau
